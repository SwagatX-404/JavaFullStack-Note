static variable
================================================================
    
    1) A variable created at class level with static keyword is called as static variable.
    2) Static variable is used for storing a values common all objects of a class 
        and common all methods of this class.
    3) static variable will have only one copy memory
    4) static variable will automatically get memory when its class is loaded into JVM.

Non-static variable
================================================================

    1) A variable created at class level without using static kw
    2) NSV is used for storing a value specific to each and every object separetly.
    3) NSV will have mulitple copies of memory 1 per each obj
    4) NSV will get memory only when we create object
    5) NSV will get mul cps of mem, as many objs as we cretd
